## Project Overview
This project applies Python data science tools to the Kaggle Titanic dataset, covering:
- Data acquisition and inspection
- Data cleaning (missing values, duplicates)
- Data visualisation (histograms, bar charts, boxplots, scatter plots, heatmap, pairplot)
- Statistical analysis (descriptive statistics, frequency distributions, correlation analysis)
- Machine learning (Logistic Regression classifier predicting passenger survival)

## How to Run
1. Clone this repository.
2. Install dependencies:
   ```
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```
3. Launch Jupyter Notebook:
   ```
   jupyter notebook Titanic_Data_Science_Project.ipynb
   ```
4. Run all cells (Kernel > Restart & Run All).

## Dataset Source
Kaggle Titanic Competition: https://www.kaggle.com/competitions/titanic/data
(`train.csv` used for this project, licensing permits redistribution for educational use.)

## Key Results
- Logistic Regression model achieved **~81% accuracy** predicting passenger survival.
- Strongest predictors of survival: passenger sex, passenger class, and fare paid.
- Full discussion of findings, limitations, and recommendations available in the report and
  in Task 6 of the notebook.
